Primeira Versión da WebApp 2026

Trátase dun arquivo HTML que pode funcionar perfectamente en web ou Movil.

CARACTERÍSTICAS:
- HTML e JavaScript
- Responsive

QUE FAI?
- Suma automaticamente a puntuación por dependencia en pista dos xogadores e xogadoras dun equipo, avisando si se sobrepasa o límite permitido.
- Incorpora opcións de: Dorsal, Player Femenino-Masculino, Menor de 23 anos.
- Opción de xogadora de género Feminino.
- Opción de modo Oscuro-Claro.
- Opción de Gardar e Cargar un quinteto predeterminado.
- Contral de equipo Local e Visitante.
